# 用b站直播弹幕控制游戏

## 使用方法
1. 开启游戏
2. 在聊天栏中输入 `roomid <房间id>`
3. 去对应的直播间刷弹幕吧
4. 也可以直接在聊天栏输入弹幕指令测试

** 注意:如果是大主播的话，房间号可能不是连接后的数字,需要进行查询一次
https://api.live.bilibili.com/room/v1/Room/room_init?id=<url中的id>
